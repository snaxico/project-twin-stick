## How to Use

1. Open `Sprite_Context_Short.md` and follow the setup (rules + reference image)
2. Pick a template below and copy it
3. Paste into ChatGPT
4. Download and follow `Sprite_Generation_Workflow.md`
5. Repeat for the next sprite

## Generation Tips

- **One chat = ~5–10 sprites** before quality drifts
- **Always paste the reference image URL** at the start of each chat
- **One sprite at a time** — don't batch requests
- **Background removal is manual** — use remove.bg, GIMP, or Photoshop
- **Resize with nearest-neighbor** to preserve crisp edges
- **Test each sprite in Godot** before approval

See `Sprite_Generation_Workflow.md` for the complete 6-phase process.
