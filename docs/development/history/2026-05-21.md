## Design Decisions Locked

### Tier 3: Side Objectives + Temp Buffs
- Simplified from 3 side objective types to **Hold Zone only**.
- Hold Zone: stand in a marked zone for 10 cumulative seconds (timer pauses when no player is present, enemies don't affect timer, co-op timer not per-player).
- Completing the objective rewards a random temporary buff (speed +50%, damage +50%, or attack speed +50%) shared across all living players.
- Buff lasts until room end (persists through death/revive within same room, does NOT persist between rooms).

### Tier 4: Elite Rare Mutation Delivery
- Elite rooms offer exactly **1 rare mutation** at room-end (buy or skip). No common mutations in elite rooms.
- Combat rooms continue to offer commons only (buy 0–3 picks).
- `ELITE_RARE_PICK_COST := 50` (placeholder).

### Tier 5: Encounter Depth
- **3 minor modifiers** (all +0.15x gold): `accelerating_waves`, `enemy_faster`, `spitter_swarm`.
- **3 major modifiers** (all +0.3x gold): `fire_floor`, `ice_zone`, `mine_field`.
- Minor = enemy behavior changes; Major = arena hazard changes.
- Modifiers assigned at run generation from a curated valid-combinations list.
- Combat rooms: 0–1 minor + 0–1 major. Elite rooms: 1–2 minor + 1 major. Rest/Shop/Boss: none.
- Gold multipliers are additive (not multiplicative).
- **Spitter enemy**: ranged, mid-stats between Chaser and Charger, kiting AI, 1 projectile per 0.5s.
- **Economy flatten**: all enemies drop 1g flat, survival bonus removed.

### Tier 6: Elite Room Identity
- **3 elite mini-bosses** (exclusive to elite rooms):
  - `elite_charger` (melee): 1440 HP, chaser-speed, charge+slam combo.
  - `elite_spitter` (ranged): 576 HP, 3-projectile burst, AoE energy pulse.
  - `elite_support` (support): 900 HP, aura buffs allies / debuffs players, AoE pulse damages players + heals allies.
- Each elite room: 1 guaranteed random mini-boss + standard spawn mix.
- Mini-bosses do NOT appear in combat rooms.

## Key Review Findings Addressed

- `RunState._build_single_room_map()` rewrites `elite` → `combat`, blocking elite testing (Slice 0a fix).
- `Player.apply_loadout()` bakes stats with no removal path — added reversible stat modifier layer (Slice 0b).
- `MutationSystem.roll_mutation_options()` blindly pads with wrong-rarity mutations on pool exhaustion — return fewer instead (Slice 2).
- `Enemy.apply_room_modifier()` resets health every call — new reversible `apply_aura()`/`clear_aura()` API (Slice 6).
- HUD shows Lv2 commons as 2 chips — accepted UX gap, deferred to polish.
