# 2026-05-20

## What Changed

- Implemented the first live v3 gold economy slice.
- Added per-player gold wallets to `RunState` / `PlayerInventory`.
- Added neon gold pickups that spawn from enemy deaths, magnet to the nearest player, and auto-collect on room clear.
- Added a flat room-clear survival bonus and room-level gold-earned summary text.
- Reworked the mutation reward screen from one free pick into a gold-gated multi-buy flow:
  - `3` rolled options
  - buy `0–3` picks
  - placeholder costs `15 / 50 / 100`
- Added per-player gold display to both the combat HUD and the route-map screen.

## Why

- The previous v3 loop rewarded passive survival too evenly because every cleared room granted a free mutation.
- The new slice keeps `Survive` as the room-clear rule while tying most progression to actual combat income.
- Shared pickups plus personal wallets preserve couch co-op readability without letting one player steal resources from the other.

## Design Decisions Locked (Brainstorm Session)

### Mutation Rarity Split
- **Common mutations**: upgradable (Lv1 → Lv2 → Lv3). Stat-scaling (pierce, fire rate, damage, etc). Available from all room-end picks. Picking the same one again upgrades it.
- **Rare mutations**: one-off binary effects (fire trail, ricochet, split shot, chain lightning). NOT upgradable. Only available from elite room rewards and shop purchases. Not in normal room-end pick pools.
- This gives elite rooms and shops unique identity beyond "more gold."
- Max level for commons: 3 (placeholder, balance later).

### Side Objectives Reward Temporary Buffs
- Side challenges (Hold Zone, Beacon Defense, Collection) now reward **temporary buffs** instead of extra gold.
- Buffs last **until room end** (not a fixed timer). Completing early = more buff time = more value.
- Buff types: speed boost (Hold Zone), damage boost (Beacon Defense), attack speed boost (Collection).
- Buffs are shared across all players.
- This creates a clean three-layer reward stack with no overlap:
  - Temp buffs = in-room, from side objectives
  - Common mutations = cross-room scaling, from room-end picks
  - Rare mutations = build-defining, from elites/shops

### Gold Economy Balance
- Noted: mutations are currently too cheap. All gold/cost values remain placeholder and need a dedicated balance pass after playtesting.
